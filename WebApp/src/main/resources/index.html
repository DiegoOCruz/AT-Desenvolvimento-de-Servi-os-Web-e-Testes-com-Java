Relatório de Desenvolvimento da Aplicação Web

Introdução

O presente relatório destina-se a fornecer uma visão geral do desenvolvimento da aplicação web, destacando os principais aspectos técnicos, funcionalidades implementadas e desafios enfrentados durante o processo.

Objetivo

O objetivo principal do projeto foi criar uma aplicação web robusta e funcional que permita aos usuários realizar operações relacionadas a clientes, produtos, pedidos e endereços de forma eficiente e intuitiva.

Tecnologias Utilizadas

  • Java: Linguagem de programação principal para o desenvolvimento da aplicação.
  • Spark Framework: Utilizado para o desenvolvimento de APIs RESTful.
  • Gson: Biblioteca para serialização e desserialização de objetos Java em JSON.
  • HTML/CSS: Utilizados para a criação das interfaces de usuário.
  • Git: Utilizado para controle de versão do código fonte.
  • GitHub: Repositório remoto para hospedagem do código fonte.
  • Maven: Utilizado para gerenciamento de dependências e construção do projeto.

Funcionalidades Implementadas

  • Clientes
    • Cadastro, exclusão e consulta de clientes.
    • Associação de endereços aos clientes.
  • Produtos (Bebidas e Comestíveis)
    • Cadastro, exclusão e consulta de bebidas e comestíveis.
    • Diferenciação entre tipos de bebidas (alcoólicas e não alcoólicas).
    • Cálculo automático do preço de venda dos produtos.
  • Pedidos
    • Cadastro, exclusão e consulta de pedidos.
    • Adição e remoção de produtos aos pedidos.
    • Cálculo automático do valor total do pedido.
  • Endereços
    • Cadastro, exclusão e consulta de endereços.
    • Consulta de endereço por CEP utilizando serviço externo.

Desafios Enfrentados

  • Integração com serviço externo de consulta de endereços por CEP.
  • Implementação de lógica para cálculo do preço de venda dos produtos.
  • Garantia da consistência e integridade dos dados ao associar clientes e endereços.

Funcionamento da Aplicação

A aplicação foi desenvolvida utilizando Java em conjunto com o Spark Framework, uma biblioteca Java para criação de aplicativos web. Ela permite o gerenciamento eficiente de clientes, produtos e pedidos em uma loja online. Aqui está um resumo do seu funcionamento:

  • Gerenciamento de Clientes:
    • Cadastro
    • Consulta
    • Exclusão
  • Gerenciamento de Produtos:
    • Cadastro
    • Consulta
    • Exclusão
  • Gerenciamento de Pedidos:
    • Criação
    • Cálculo Automático
    • Consulta

Integração com API Externa

A aplicação faz uso de uma API externa para melhorar sua funcionalidade e oferecer recursos adicionais aos usuários.

  • API de Consulta de Endereço por CEP - https://viacep.com.br/
