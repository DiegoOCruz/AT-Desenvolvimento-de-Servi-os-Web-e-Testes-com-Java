Introdução

Bem-vindo à documentação da API AT (Alimentos e Bebidas), uma API para gerenciamento de pedidos, clientes, bebidas, comidas e endereços. Esta API fornece endpoints para listar, adicionar, atualizar e excluir informações relacionadas a clientes, pedidos, bebidas, comidas e endereços.

Visão Geral

A API AT possui os seguintes recursos principais:

  • Cliente: Gerenciamento de informações de clientes.
  • Bebidas: Gerenciamento de bebidas disponíveis para pedido.
  • Pedidos: Gerenciamento de pedidos de clientes.
  • Comidas: Gerenciamento de comidas disponíveis para pedido.
  • Endereço: Gerenciamento de informações de endereços.

Recursos

Cliente

Listar Clientes

Endpoint: GET /cliente/obterLista

Descrição: Retorna a lista de todos os clientes cadastrados.

Listar Cliente por ID

Endpoint: GET /cliente/{id}/obterLista

Descrição: Retorna as informações de um cliente específico com base no ID fornecido.

Excluir Cliente

Endpoint: DELETE /cliente/{id}/excluir

Descrição: Exclui um cliente específico com base no ID fornecido.

Incluir Cliente

Endpoint: POST /cliente/incluir

Descrição: Adiciona um novo cliente com as informações fornecidas no corpo da solicitação.
